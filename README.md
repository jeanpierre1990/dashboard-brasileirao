# ⚽ Dashboard Interativo – Brasileirão Série A (03-24)

Um dashboard profissional desenvolvido em **Python + Streamlit**, com visual moderno, múltiplas abas, gráficos interativos e análise completa de clubes, estádios, técnicos e desempenho ao longo das temporadas.

Este projeto transforma dados do Brasileirão em uma experiência visual rica, ideal para estudos, portfólio, análise esportiva e demonstração de habilidades em Data Science.

---

## 🚀 Funcionalidades

### 🏆 **1. Ranking de Clubes**
- Tabela completa com:
  - Jogos
  - Gols pró / contra
  - Saldo de gols
  - Chutes, escanteios, faltas, defesas, impedimentos
- Escudo do clube exibido automaticamente
- Gráfico de barras do saldo de gols
- Radar interativo de estatísticas

---

### 🏟️ **2. Análise de Clubes**
- Comparação entre dois clubes
- Escudos lado a lado
- Radar comparativo
- Tabela de métricas agregadas

---

### 📍 **3. Estádios**
- Ranking de estádios por jogos e gols
- Mapa interativo com coordenadas reais
- Tooltip com estatísticas
- Estrutura pronta para fotos dos estádios

---

### 🧑‍🏫 **4. Técnicos**
- Ranking de técnicos por desempenho
- Comparação entre dois treinadores
- Gráficos de barras agrupadas

---

### 📈 **5. Linha do Tempo**
- Evolução rodada a rodada por clube
- Gráfico interativo de gols pró

---

### 🔥 **6. Heatmap de Estatísticas**
- Heatmap normalizado por clube
- Comparação visual rápida de desempenho

## ▶️ Como Executar

1. Instale as dependências:

```bash
pip install streamlit pandas plotly
````
2. Execute o dashboard:
```
streamlit run dashboard.py
```
3. O navegador abrirá automaticamente em:
```
http://localhost:8501
```
🧭 Navegação do Dashboard
- 🏆 Ranking – visão geral dos clubes
- 🏟️ Clubes – comparação entre equipes
- 📍 Estádios – mapa e estatísticas
- 🧑‍🏫 Técnicos – análise de treinadores
- 📈 Linha do tempo – evolução por rodada
- 🔥 Heatmap – visão geral de desempenh
